Métodos de arreglos

array.unshift

Agrega un elemento en la posición 0 de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB" ]

array_ejemplo.unshift( "stringC" );

console.log( array_ejemplo );

Obtendremos por consola: [ "stringC", "stringA", "stringB" ]

array.shift

Elimina el elemento que se encuentre en la posición 0 de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

array_ejemplo.shift();

console.log( array_ejemplo );

Obtendremos por consola: [ "stringB", "stringC" ]

array.pop

Elimina el elemento que se encuentra en la ultima posición de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

array_ejemplo.pop();

console.log( array_ejemplo );

Obtendremos por consola: [ "stringA", "stringB" ]

array.splice

Cambia el contenido de un array eliminando elementos existentes y/o agregando nuevos elementos.

Como primer parametro recibe el indice del rango del elemento o los elementos a eliminar y/o reemplazar.
y como segundo parametro la cantidad de elementos a eliminar a partir del indice indicado anteriormente.

Puede recibir como parametro el elemento a reemplazar por el eliminado.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC", "stringD", "stringE" ]

array_ejemplo.splice(1,4);

console.log( array_ejemplo );

Obtendremos por consola: [ "stringA" ]

Ejemplo enviando elemento a reemplazar:

let array_ejemplo = [ "stringA", "stringB", "stringC", "stringD", "stringE" ]

array_ejemplo.splice(2, 2, "reemplazo1", "reemplazo2");

console.log( array_ejemplo );

Obtendremos por consola: [ 'stringA', 'stringB', 'reemplazo1', 'reemplazo2', 'stringE' ]

array.join

Devuelve un string con todos los elementos del arreglo indicado.

No modifica nuestro arreglo, hace una copia del mismo.

Puede recibir como parametro el divisor que vamos a emplear entre nuestro strings.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

console.log( array_ejemplo.join() );

Obtendremos por consola: stringA,stringB,stringC,stringD,stringE

Ejemplo indicando divisor:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

console.log( array_ejemplo.join(" - ") );

Obtendremos por consola: stringA - stringB - stringC - stringD - stringE

array.concat

Concatena dos arreglos. No los modifica.

Devuelve un nuevo arreglo con el resultado de la concatenación.

Ejemplo:

let array_ejemplo1 = [ "stringA", "stringB" ]

let array_ejemplo2 = [ 10, 20 ]

console.log( array_ejemplo1.concat( array_ejemplo2 ) );

Obtendremos por consola: [ "stringA", "stringB", 10, 20 ]

array.slice

Retorna un nuevo array cortando el array indicado con los indices indicados del mismo.

Recibe dos parametros ( indice 1 e indice 2 ). Donde indice 1 indica el inicio del corte, e indice 2 indica el final del corte.

No abarca el ultimo elemento señalado por el indice 2 al retornar.
Es decir, retorna un array que comienza desde el primer elemento señalado, hasta el ante ultimo elemento señalado por los indices.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.slice( 2, 4 ) );

Obtendremos por consola: [ "stringC", 10 ]

array.indexOf

Nos permite conocer el indice de un elemento en nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.indexOf( "StringC" ) );

Obtendremos por consola: 2

array.reverse

Nos permite revertir el orden de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.reverse( ) );

Obtendremos por consola: [ 20, 10, "stringC", "stringB", "stringA" ]
