Métodos de arreglos

array.unshift

Agrega un elemento en la posición 0 de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB" ]

array_ejemplo.unshift( "stringC" );

console.log( array_ejemplo );

Obtendremos por consola: [ "stringC", "stringA", "stringB" ]

array.shift

Elimina el elemento que se encuentre en la posición 0 de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

array_ejemplo.shift();

console.log( array_ejemplo );

Obtendremos por consola: [ "stringB", "stringC" ]

array.pop

Elimina el elemento que se encuentra en la ultima posición de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

array_ejemplo.pop();

console.log( array_ejemplo );

Obtendremos por consola: [ "stringA", "stringB" ]

array.splice

Cambia el contenido de un array eliminando elementos existentes y/o agregando nuevos elementos.

Como primer parametro recibe el indice del rango del elemento o los elementos a eliminar y/o reemplazar.
y como segundo parametro la cantidad de elementos a eliminar a partir del indice indicado anteriormente.

Puede recibir como parametro el elemento a reemplazar por el eliminado.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC", "stringD", "stringE" ]

array_ejemplo.splice(1,4);

console.log( array_ejemplo );

Obtendremos por consola: [ "stringA" ]

Ejemplo enviando elemento a reemplazar:

let array_ejemplo = [ "stringA", "stringB", "stringC", "stringD", "stringE" ]

array_ejemplo.splice(2, 2, "reemplazo1", "reemplazo2");

console.log( array_ejemplo );

Obtendremos por consola: [ 'stringA', 'stringB', 'reemplazo1', 'reemplazo2', 'stringE' ]

array.join

Devuelve un string con todos los elementos del arreglo indicado.

No modifica nuestro arreglo, hace una copia del mismo.

Puede recibir como parametro el divisor que vamos a emplear entre nuestro strings.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

console.log( array_ejemplo.join() );

Obtendremos por consola: stringA,stringB,stringC,stringD,stringE

Ejemplo indicando divisor:

let array_ejemplo = [ "stringA", "stringB", "stringC" ]

console.log( array_ejemplo.join(" - ") );

Obtendremos por consola: stringA - stringB - stringC - stringD - stringE

array.concat

Concatena dos arreglos. No los modifica.

Devuelve un nuevo arreglo con el resultado de la concatenación.

Ejemplo:

let array_ejemplo1 = [ "stringA", "stringB" ]

let array_ejemplo2 = [ 10, 20 ]

console.log( array_ejemplo1.concat( array_ejemplo2 ) );

Obtendremos por consola: [ "stringA", "stringB", 10, 20 ]

array.slice

Retorna un nuevo array cortando el array indicado con los indices indicados del mismo.

Recibe dos parametros ( indice 1 e indice 2 ). Donde indice 1 indica el inicio del corte, e indice 2 indica el final del corte.

No abarca el ultimo elemento señalado por el indice 2 al retornar.
Es decir, retorna un array que comienza desde el primer elemento señalado, hasta el ante ultimo elemento señalado por los indices.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.slice( 2, 4 ) );

Obtendremos por consola: [ "stringC", 10 ]

array.indexOf

Nos permite conocer el indice de un elemento en nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.indexOf( "StringC" ) );

Obtendremos por consola: 2

array.reverse

Nos permite revertir el orden de nuestro array.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.reverse( ) );

Obtendremos por consola: [ 20, 10, "stringC", "stringB", "stringA" ]

Métodos de arreglos basados en High Order Functions

array.find

El método find() recibe una función de comparación por parámetro (es decir, una función que captura el elemento que se está recorriendo y retorna true o false según la comparación que ejecuta).

El método retorna el primer elemento del array que cumpla con esa condición.

Ejemplo:

let array_ejemplo = [ "stringA", "stringB","stringC", 10, 20 ]

console.log( array_ejemplo.find( function(e) { return e === "stringB" } ) );

Obtendremos por consola: stringB ;

array.forEach

El método forEach() Itera sobre el array de referencia y por cada elemento del array ejecuta la función que enviemos por parámetro, la cual recibe a su vez el elemento del array que se está recorriendo.

Ejemplo:

let array_ejemplo = [1, 2, 3, 4, 5, 6]

array_ejemplo.forEach( function( n ) { console.log( n ) } );

Obtendremos por consola:

1
2
3
4
5
6

array.some

El método some() funciona igual que el método find(), recibiendo una función de búsqueda, con la diferencia que en vez de retornar el elemento encontrado, el método some retorna true o false según el resultado de la iteración de búsqueda.

Ejemplo:

let array_ejemplo = [ { nombre: "Javascript", precio: 15000 },
{ nombre: "ReactJS", precio: 22000 },
{ nombre: "AngularJS", precio: 22000 },
{ nombre: "Desarrollo Web", precio: 16000 }, ];

console.log( array_ejemplo.some( function( c ) { return c.nombre === "Desarrollo Web" } ) );

Obtendremos por consola: true;

array.map

El método map() se utiliza para crear un nuevo array con todos los elementos del array original, transformados según las operaciones de la función enviada por parámetro.

El nuevo array obtenido tiene la misma cantidad de elementos que el array original, pero los elementos que se almacenan son el return de la función enviada por parámetro.

Ejemplo:

let array_ejemplo = [ { nombre: "Javascript", precio: 15000 },
{ nombre: "ReactJS", precio: 22000 },
{ nombre: "AngularJS", precio: 22000 },
{ nombre: "Desarrollo Web", precio: 16000 }, ];

console.log( array_ejemplo.map( function( c ) { return c.nombre } ) );

Obtendremos por consola: [ "Javascript", "ReactJS", "AngularJS","Desarrollo Web" ]

array.reduce

El método reduce() nos permite obtener un único valor tras iterar sobre el array. Es decir, funciona como un método que resume el array a un único valor de retorno.

Se utiliza mucho cuando, por ejemplo, queremos acumular la suma de alguna propiedad de los elementos almacenados, u obtener algún resultado general sobre todo el array.

A diferencia de los métodos anteriores, el método reduce recibe dos parámetros. El primero es la función que ordena qué queremos resumir del array, y recibe primero un parámetro que funciona como acumulador y el segundo es el elemento del array que iteramos.

Ejemplo:

let array_ejemplo = [ { nombre: "Javascript", precio: 15000 },
{ nombre: "ReactJS", precio: 22000 },
{ nombre: "AngularJS", precio: 22000 },
{ nombre: "Desarrollo Web", precio: 16000 }, ];

console.log( array_ejemplo.reduce( function( acumulador, curso ) { return acumulador + curso.precio }, 0 );

Obtendremos por consola: 75000

array.sort

El método sort() nos permite reordenar un array según un criterio que definamos.

Este método es destructivo, es decir que modifica el array sobre el cual se llama.

El método recibe una función de comparación por parámetro que recibe dos elementos del array a la vez.

La función debe retornar un valor numérico (1, -1, 0) que indica qué elemento se posiciona antes o después. Para ordenar números, basta con restar uno al otro, y el orden indica si será ordenado de forma ascendente o descendente.

Ejemplo:

let array_ejemplo = [ 40, 1, 5, 200 ]

console.log( array_ejemplo.sort( function( a, b ) { return a - b } );

Obtendremos por consola: [ 1, 5, 40, 200 ]

console.log( array_ejemplo.sort( function( a, b ) { return b - a } );

Obtendremos por consola: [ 200, 40, 5, 1 ]
